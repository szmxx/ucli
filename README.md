## Usage
Install package:
```bash
# npm
npm install -g @szmxx/ucli

# yarn
yarn add -g @szmxx/ucli

# pnpm
pnpm install -g @szmxx/ucli
```

Add authentication information
```bash
git config --global ucli.auth <github_auth>
```

Create a new project
```bash
# create
ucli create <project-name>
# cd project
cd <project-name>
# start project
pnpm dev
```
